critters

Procedural creature sandbox: primitive shapes fused into seamless toon bodies by an SDF blend-shell vertex shader, animated 100% procedurally. One HTML file, no assets. Source on GitHub.

Credits

Original demo + technique: GOOBERS (seamless blend shell critters) by u/AntiqueFeedback7447: Reddit post (r/aigamedev, July 2026) and X post. They published the technique writeup and their prompt; their code was never released. Everything here was written from scratch against that public description plus screenshots. Disclosed in the thread as u/MTOMalley.
